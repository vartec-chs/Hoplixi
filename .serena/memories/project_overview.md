Hoplixi is a cross-platform Flutter password manager with desktop and mobile targets. Main stack: Flutter/Dart, Riverpod (without riverpod_generator), typed_prefs over SharedPreferences/flutter_secure_storage, Drift + SQLite3 Multiple Ciphers for encrypted store data, GetIt for DI, GoRouter for navigation, flutter_rust_bridge for Rust integration. Root entrypoint is lib/main.dart; features are organized under lib/features, core app services/utilities under lib/core, and encrypted database logic under lib/main_store. AGENT.md is the primary project-specific instruction source; docs-ai/* supplements it.
