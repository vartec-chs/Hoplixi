# Hoplixi v1.2.5

Дата релиза: 2026-05-04

## Что нового

### Password Manager

- В настройках хранилища добавлена отдельная страница для включения и отключения
  device key через проверку текущего мастер-пароля и rekey.
- Диалоги выбора и сохранения JSON key file переведены на расширение `.hdbkey`.
- Обновлены потоки выбора и генерации key file под новый суффикс и общую
  константу расширения.
- В модалке настроек появились отдельные переходы к страницам JSON key file и
  device key.

### Security / KDF

- Для текущих хранилищ унифицирован путь деривации ключа: create, open и rekey
  используют одну и ту же KDF-конфигурацию из `StoreKeyConfig`.
- Контекстная деривация для v2 теперь покрывает основные сценарии работы с
  ключом хранилища и дополнительными факторами.

### Store Settings UX

- Добавлена отдельная страница для device key с понятной индикацией состояния,
  полем мастер-пароля и действиями включения/отключения.
- Обновлена документация фичи `store_settings` под новый flow.

## Важно перед обновлением

- Если у хранилища включён JSON key file, перед переключением device key нужно
  выбрать текущий key file на соседней странице.
- При включённом device key хранилище будет открываться только на том
  устройстве, для которого был выполнен rekey.

## Для разработчиков

- `_deriveV2ContextKey` стал основной веткой для `kdfVersion = 2`.
- `StoreSettingsState` и `StoreSettingsNotifier` расширены полями и методами для
  управления device key и JSON key file rekey flow.
- Добавлена отдельная UI-страница `DeviceKeySecuritySection`.
