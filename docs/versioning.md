# Нумерация Версий

Версии Hoplixi нумеруются в формате `X.Y.Z`.

- `X` - глобальные изменения, затрагивающие ядро БД, таблицы, сущности и
  близкие архитектурные части.
- `Y` - изменения, требующие миграции базы данных.
- `Z` - все остальные изменения: UI, UX, документация, исправления и прочие
  доработки без миграции БД.

Если изменение затрагивает несколько уровней сразу, выбирается более старший
разряд. Например, миграция схемы поднимает `Y`, даже если вместе с ней есть
дополнительные UI-правки.
