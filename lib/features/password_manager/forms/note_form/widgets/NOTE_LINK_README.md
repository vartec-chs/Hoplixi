# Note Link Button - Ссылки на vault items в Quill Editor

## Описание

Кастомная кнопка для Quill Editor, которая позволяет вставлять ссылки на любые
vault items. При клике на кнопку открывается модальное окно выбора объекта, после
чего вставляется кликабельная ссылка.

## Возможности

✅ Кастомная кнопка в Quill Toolbar ✅ Интеграция с модальным окном выбора
объектов ✅ Вставка ссылок в формате `vault://<entityType>/<itemId>` ✅
Чтение legacy ссылок `note://noteId` ✅ Обработка кликов по ссылкам с
предложением открыть связанную запись ✅ Сохранение названия объекта в тексте
ссылки

## Как работает

### 1. Вставка ссылки

```dart
// При нажатии на кнопку:
1. Открывается модалка выбора объекта
2. Пользователь выбирает vault item
3. В позицию курсора вставляется текст с именем объекта
4. К тексту применяется ссылка формата: vault://entityType/itemId
```

### 2. Клик по ссылке

```dart
// При клике на внутреннюю ссылку:
1. Проверяется формат URL (`vault://...` или legacy `note://...`)
2. Извлекается ID связанного объекта
3. Показывается диалог подтверждения
4. При подтверждении открывается связанная запись
```

## Формат ссылок

Основной формат ссылок:

```
vault://entityType/itemId
```

Legacy формат `note://noteId` продолжает поддерживаться для обратной совместимости.

## Интеграция в Toolbar

В `note_form_screen.dart` кнопка добавлена рядом с стандартными кнопками Quill:

```dart
Row(
  children: [
    QuillSimpleToolbar(...),
    NoteLinkButton(
      controller: _quillController,
      iconSize: 18,
    ),
  ],
)
```

## Кастомизация

### Изменить иконку

```dart
NoteLinkButton(
  controller: _quillController,
  iconSize: 20,  // Размер иконки
  iconTheme: QuillIconTheme(
    iconUnselectedColor: Colors.blue,
    iconUnselectedFillColor: Colors.white,
  ),
)
```

## Компоненты

### NoteLinkButton

Кастомная кнопка для toolbar:

- Открывает модалку выбора объекта
- Вставляет ссылку в документ
- Поддерживает форматирование выделенного текста

### onLaunchUrl handler

В QuillEditorConfig добавлен обработчик:

- Перехватывает клики по ссылкам
- Проверяет формат `vault://` и legacy `note://`
- Показывает диалог открытия связанной записи

## Сценарии использования

### Создание ссылки на пустом месте

1. Установите курсор в нужное место
2. Нажмите кнопку ссылки
3. Выберите объект из списка
4. Текст с именем объекта будет вставлен и отформатирован как ссылка

### Превращение текста в ссылку

1. Выделите существующий текст
2. Нажмите кнопку ссылки
3. Выберите объект
4. Выделенный текст станет ссылкой на объект

### Открытие связанной записи

1. Кликните на внутреннюю ссылку в тексте
2. Подтвердите открытие в диалоге
3. Связанная запись откроется в режиме редактирования

## Технические детали

### LinkAttribute

Используется стандартный `LinkAttribute` из flutter_quill:

```dart
controller.formatText(
  index,
  length,
  LinkAttribute(linkUrl),
);
```

### Навигация

При открытии связанной записи используется `go_router`:

```dart
context.pushNamed(
  'entity_edit',
  pathParameters: {'entity': entityType.id, 'id': itemId},
);
```
